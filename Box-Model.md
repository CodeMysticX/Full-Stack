# Box-Model

The following HTML code demonstrates the box model concept in CSS:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <title>Box-Model</title>
    <style>
        .container{
            border: 10px dashed black;
            padding: 25%;
            margin: 20%;
            position: absolute;
        }
    </style>
</head>
<body>
    <div class="container">this is inside div</div>
</body>
</html>
